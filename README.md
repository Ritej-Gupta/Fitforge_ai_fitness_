# Project Submission: FitForge AI 🏋️‍♂️

**Challenge:** Food & Health App  
**Project Name:** FitForge AI - The Intelligent Fitness Hub

## 📝 Project Description

FitForge AI is a comprehensive, AI-driven fitness ecosystem built to empower gym enthusiasts. By combining Google Gemini 1.5 Flash with Firebase Firestore, we have created a single-entry platform that handles the complex math and science of fitness so the user can focus on the heavy lifting.

The app features a cinematic, low-opacity animated background of rotating gym equipment to provide an immersive ideathon feel without sacrificing performance.

## 🛠 Key Updates in Deployed Version

- **Unified AI Engine:** Gemini 1.5 Flash for natural language calorie estimation and personalized Indian meal/workout generation.
- **Cinematic UI/UX:** Custom HTML5 Canvas background animation for high visual appeal and modern aesthetic.
- **Persistence Layer:** Real-time data syncing via Firebase Firestore for weight and caloric tracking.
- **Responsive Architecture:** Optimized for mobile-first usage (standard for gym environments).
- **Security Protocols:** Strict input sanitization and secure API request handling.

## 🧠 Core Features

- 📊 **Intelligent Dashboard:** Real-time TDEE/BMI calculations and goal tracking.
- 💪 **Workout AI:** Generates hyper-specific split routines (PPL, Upper/Lower) based on user goals.
- 🥗 **Indian Meal AI:** Localized diet planning using Indian ingredients and macro-mapping.
- 🔥 **AI Calorie Tracker:** Converts natural language (e.g., "3 rotis and 1 bowl of dal") into numeric macro data.
- 📈 **Weight Tracker:** Visual progress monitoring using Chart.js integration.
- 💊 **Supplement Lab:** Scientific breakdown of dosages and benefits for common fitness supplements.
